### Info

replica of [seq-docker-windows](https://github.com/datalust/seq-docker-windows)
one can use Structured Sinks built in __Seq__, which natively visualize and query structured data without needing extra formatting code that is required with Console or File sinks (`Serilog.Sinks.Console` or `Serilog.Sinks.File`)
