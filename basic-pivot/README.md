### Info

[Apache pivot](https://en.wikipedia.org/wiki/Apache_Pivot) [Widget Toolkit](https://en.wikipedia.org/wiki/Apache_Pivot#WTK) (__WTK__) file open dialog example for local file system access to an application

NOTE> This project has retired
### Usage

```sh
mvn package
```
```sh
java -cp target\example.pivot.jar;target\lib\* example.App
```
